-Objetivo del sistema

El objetivo de este proyecto es desarrollar un Generador Seguro de Contraseñas que permita crear claves aleatorias utilizando letras mayúsculas, letras minúsculas y números. El sistema busca ayudar a los usuarios a mejorar la seguridad de sus cuentas mediante la generación de contraseñas más difíciles de adivinar.

-Descripción del problema:

Actualmente muchas personas utilizan contraseñas débiles, como nombres propios, fechas de nacimiento o combinaciones muy simples. Esto incrementa el riesgo de que sus cuentas sean vulneradas por terceros.

Para aportar una solución a este problema, se desarrolló un sistema que genera contraseñas aleatorias con diferentes longitudes, proporcionando una alternativa más segura para proteger la información personal de los usuarios.

-Funcionalidades del sistema

El sistema desarrollado permite realizar las siguientes acciones:
Ingresar la longitud de la contraseña (entre 1 y 100 caracteres),
Validar que el dato ingresado sea correcto,
Generar una contraseña aleatoria utilizando letras mayúsculas, letras minúsculas y números,
Mostrar la contraseña generada al usuario,
Permitir generar nuevas contraseñas sin necesidad de reiniciar el programa.
