Objetivo del sistema:
Desarrollar un generador seguro de contraseñas que permita crear claves aleatorias utilizando letras mayúsculas, minúsculas y números, con el propósito de mejorar la seguridad de las cuentas de los usuarios.

--Descripción de funcionalidades--

El sistema permite:
Ingresar la longitud de la contraseña (entre 1 y 100 caracteres).
Validar que la longitud ingresada sea correcta.
Generar una contraseña aleatoria utilizando mayúsculas, minúsculas y números.
Mostrar la contraseña generada en pantalla.
Permitir al usuario generar una nueva contraseña si lo desea.
